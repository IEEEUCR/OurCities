Welcome to OurCities, a collaborative website for pages written by members of IEEE@UCR. Contributing pages is easy!

  1. Visit the IEEEUCR/OurCities repository
  2. Fork the repository
  3. Create a new HTML page (Copy this page and add your own text!)
  4. Commit the changes to your repository
  5. Create a Pull Request at IEEE/OurCities

Contribute your page today!
